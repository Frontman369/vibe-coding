# vibe-coding

A growing collection of experiments, tools, apps, and weird ideas built through AI-assisted development.

This repo is where I ship projects created with tools like Replit, AI coding agents, LLMs, and whatever else helps turn an idea into working software fast. Some projects are polished, some are chaotic prototypes, and some exist purely because “what if?” felt interesting enough to build.

The goal isn’t just speed — it’s exploration.

Inside this repository, you’ll find:

* Small utilities solving real problems
* Web apps and experiments
* Security-focused tools and learning projects
* Automation ideas
* Interface concepts
* Random late-night builds that somehow worked

Each project represents a snapshot of modern “vibe coding”:
human direction + AI collaboration + rapid iteration.

I use this repo as:

* A public build log
* A playground for ideas
* A way to document progress over time
* A reminder that shipping beats overthinking

Some projects may evolve into standalone repositories later. Others will stay here as artifacts of experimentation and learning.

Tech stacks will vary depending on the project, but expect things like:

* TypeScript / JavaScript
* React
* Node.js
* Python
* AI APIs
* Replit deployments
* Local-first and privacy-focused concepts

Nothing here is guaranteed to be perfect. Everything here is guaranteed to be built.

Welcome to the lab.
